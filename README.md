# NexusMesh
A distributed Ai-powered eco-physiological sensing platform that protects workers and community in real time.
"We built an intelligent workplace safety system that detects fevers AND gas leaks simultaneously. Using an infrared sensor and MQ-2 gas sensor, it monitors body temperature and air quality. What makes it smart? It learns each person's normal temperature over time and alerts only when deviations matter. Gas detection triggers immediate evacuation alerts. All processing happens locally on an ESP32 - no cloud, no delay, no privacy concerns. Perfect for workers in hazardous locations and research"
